# Simple Arcade Games

A small collection of browser-based arcade games built with HTML, CSS, and vanilla JavaScript. The focus is smooth gameplay, clean visuals, and lightweight code you can extend.

## Games
- Neon Stacks: a modern Tetris-inspired block stacking game with level-based speed.

## Features
- HTML5 Canvas rendering
- Level progression with increasing drop speed
- Score, lines, and combo tracking
- Modern neon-inspired UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
1. Open `index.html` in a modern browser.
2. Press Enter or click Start to begin.

Tip: You can use any local static server if you prefer (for example, VS Code Live Server).

## Controls
- Move: Arrow Left / Arrow Right
- Soft drop: Arrow Down
- Rotate: Arrow Up / Z / X
- Hard drop: Space
- Pause/Resume: P / Enter

## Project Structure
- `index.html` - layout and UI shell
- `styles.css` - visual theme and layout styling
- `main.js` - gameplay logic and rendering
- `plan.md` - project planning notes
