# Simple Arcade Games

A small collection of browser-based arcade games built with HTML, CSS, and vanilla JavaScript. The focus is smooth gameplay, clean visuals, and lightweight code you can extend.

## Games
- Neon Stacks: a modern Tetris-inspired block stacking game with level-based speed.
- Contra Clash: a Contra-inspired combat arena with combo chains and blaster shots.

## Features
- HTML5 Canvas rendering
- Level progression with increasing drop speed
- Score, lines, and combo tracking
- Combo combat routes with momentum-based chaining
- Modern neon-inspired UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)

## How to Run
- Open `Tetris/index.html` in a modern browser for Neon Stacks.
- Open `ContraCombat/index.html` in a modern browser for Contra Clash.

Tip: You can use any local static server if you prefer (for example, VS Code Live Server).

## Controls
- **Neon Stacks** Move: Arrow Left / Arrow Right; Soft drop: Arrow Down; Rotate: Arrow Up / Z / X; Hard drop: Space; Pause/Resume: P / Enter.
- **Contra Clash** Move: A / D or Arrow Keys; Jump: W or Up; Light: J; Heavy: K; Blaster: L; Pause: Enter or P.

## Project Structure
- `Tetris/index.html` - layout and UI shell
- `Tetris/styles.css` - visual theme and layout styling
- `Tetris/main.js` - gameplay logic and rendering
- `Tetris/plan.md` - project planning notes
- `ContraCombat/index.html` - layout and HUD for the combat arena
- `ContraCombat/styles.css` - Contra-inspired UI theme
- `ContraCombat/main.js` - combat logic, combos, and rendering
